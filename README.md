# Ecommerce Work Flow

This project is a **Selenium WebDriver automation framework** built using **Java**, **TestNG**, and the **Page Object Model (POM)** design pattern.  
It automates the login, product flow and logout functionality of the following test website:

ðŸ”— https://practicetestautomation.com/practice-test-login/

The framework follows **OOP principles**, generates **Extent Reports**, and automatically captures **screenshots for every page**.

---

## Tech Stack

- **Language:** Java  
- **Automation Tool:** Selenium WebDriver  
- **Test Framework:** TestNG  
- **Build Tool:** Maven  
- **Design Pattern:** Page Object Model (POM)  
- **Reporting:** Extent Reports  
- **Browser Management:** WebDriverManager
